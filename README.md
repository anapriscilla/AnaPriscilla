##
<h3>Foco atual: Evoluir para Design Engineer</h3>
 
- 🧩 **Design Systems**: arquitetura de design tokens <br> (primitivos → semânticos → componentes), componentização e versionamento de bibliotecas no Figma.
- 🌉 **Design-to-Code**: traduzir componentes do Figma em código vivo (HTML, CSS e React), <br>garantindo que o design system exista igual no design e no front-end.
- 📦 **Handoff técnico**: especificações, documentação e guidelines.
- ♿ **Acessibilidade**: contraste, legibilidade e navegação seguindo WCAG, do design à implementação.
- 🛠️ **Programação**: aprofundando conhecimento em JavaScript e evoluindo em TypeScript e React <br>para construir e manter componentes de design systems em código.
